# KukuPaste GitHub Pages Site

This folder is ready to publish as a GitHub Pages site.

## Files

- `index.html` - public product page for Lemon Squeezy verification.
- `packs-sales-sheet.pdf` - client PDF sales sheet.
- `downloads/KukuPaste_Beta_RC_Current.exe` - current beta RC installer.

## Suggested GitHub Pages setup

1. Create a new public GitHub repository, for example `kukupaste-site`.
2. Upload the contents of this folder to the repository root.
3. Open repository Settings -> Pages.
4. Select `Deploy from a branch`.
5. Branch: `main`, folder: `/root`.
6. Save and wait for GitHub Pages to publish.

The final URL will usually look like:

```text
https://YOUR-GITHUB-USERNAME.github.io/kukupaste-site/
```

Use that URL in Lemon Squeezy as `Your website`.
