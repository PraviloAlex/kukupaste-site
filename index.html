How KukuPaste works

It is a small Windows tray app for people who move the same kinds of text between apps all day.

1. Copy normally

Use Ctrl+C in any app. KukuPaste adds each copied text to the active stack.

2. Keep order

The first copied item stays first. Your next paste is predictable.

3. Paste next

Press Ctrl+Q to paste the next stack item into the current window.

4. Reuse workflows

Save common stacks as libraries and load them again when the same work returns.

Three ways to paste

KukuPaste is built around predictable paste flow: one item, a manual chooser, or the whole stack at once.

Ctrl+Q: paste next

Paste one stack item into the active window. Press again for the next item.

Hold Ctrl+Q: choose

Open the compact chooser when you want to pick from the active stack or history.

Ctrl+Shift+Q: paste all

Insert the whole stack at once: 2, 3, 5, or more saved texts, depending on your plan limit.

After paste: history

Pasted items leave the active stack and remain in local history, so you can restore them later.

Paste all formats: plain text, separated by ---, or Section 1 / Section 2 / Section 3.

Why this is not just a clipboard manager

A normal clipboard is for finding something you copied before. KukuPaste is for finishing a small text sequence now.

Normal clipboard

You open history, search for the right text, copy it again, then hope it is the right one.

KukuPaste stack

You copy the texts once, keep the order, and paste the next item whenever you press Ctrl+Q.

Copy A -> Copy B -> Copy C -> Ctrl+Q -> A -> B -> C
